# SimpleRayTracerRust

A very simple ray tracer built using Rust programming language. 
***
**To run:**
```
cargo run > image.ppm
```
The output image is in ppm format. Use a [ppm viewer](https://www.cs.rhodes.edu/welshc/COMP141_F16/ppmReader.html) to view the image.
